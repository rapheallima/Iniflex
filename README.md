# Teste Prático de Programação - Java

Este repositório contém a resolução do teste prático para a vaga de desenvolvedor. O objetivo é demonstrar conhecimentos
em Java Core, Orientação a Objetos e manipulação de coleções.

## 🚀 Tecnologias Utilizadas

- Java 17 
- Stream API para manipulação de dados
- BigDecimal para operações financeiras precisas

## 🛠️ Funcionalidades Implementadas

- Inserção e remoção de funcionários da lista.
- Agrupamento de funcionários por função utilizando `Map`.
- Cálculo de aumento salarial (10%).
- Filtragem de aniversariantes por mês específico.
- Ordenação alfabética e cálculos de média salarial.

## 📦 Como Executar

1. Clone o repositório.
2. Certifique-se de ter o JDK instalado.
3. Execute a classe `Principal.java` localizada no pacote `src`.
